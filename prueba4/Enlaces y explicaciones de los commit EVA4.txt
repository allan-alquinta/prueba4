El nuevo estilo moderniza la interfaz sin alterar la ubicación ni dimensiones originales del formulario, manteniéndolo alineado a la izquierda y del mismo tamaño.

Commit 3: Integración con Firebase Firestore

Descripción del cambio:
Se integró Firebase Firestore al proyecto para almacenar de manera remota los registros ingresados a través del formulario. Esta mejora permite guardar, consultar y eliminar actividades directamente desde Firestore, reemplazando el uso exclusivo de localStorage.

Cambios realizados:

- Se creó el archivo firebase-config.js con la configuración de Firebase proporcionada.
- Se instaló la dependencia oficial de Firebase con el comando: npm install firebase

- Se modificó el componente Formulario.js:
- Se agregó conexión con getFirestore, collection, addDoc, getDocs, deleteDoc.
- Se reemplazó el guardado en localStorage por guardado en Firestore.
- Se recuperan automáticamente los datos desde la colección "actividades".
- Se agregó la opción de eliminar directamente desde la tabla (eliminando también en Firestore).

Verificación del cambio:
Se probó en el navegador con npm start
Se visualizó la consola del navegador para confirmar el log: Firebase App initialized: [DEFAULT]

Se comprobó que los datos aparecen en la sección Firestore Database > actividades
La tabla del frontend muestra correctamente los registros obtenidos de Firestore.
